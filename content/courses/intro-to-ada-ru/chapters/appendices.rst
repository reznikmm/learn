Неограниченная версия типа
~~~~~~~~~~~~~~~~~~~~~~~~~~

Многие из приведенных выше примеров могут быть использованы для формальных неопределенных
(неограниченных) типов:

+--------------------------------------------------------------+-------------------------+
| Формальный тип                                               | Фактический тип         |
+==============================================================+=========================+
| Неопределенный неполный тип                                  | Любой тип               |
|                                                              |                         |
| **Формат**: :ada:`type T (<>);`                              |                         |
+--------------------------------------------------------------+-------------------------+
| Неопределенный неограниченный частный тип                    | Любой неограниченный тип|
|                                                              | неопределенный или      |
| **Формат**: :ada:`type T (<>) is private;`                   | определенный            |
+--------------------------------------------------------------+-------------------------+
| Неопределенный ограниченный частный тип                      | Любой тип, ограниченный |
|                                                              | или нет, неопределенный |
| **Формат**: :ada:`type T (<>) is limited private;`           | или определенный        |
+--------------------------------------------------------------+-------------------------+
| Неполный неопределенный частный тип с тегами                 | Любой конкретный тип с  |
|                                                              | тегами, неопределенный  |
| **Формат**: :ada:`type T (<>) is tagged;`                    | или определенный        |
+--------------------------------------------------------------+-------------------------+
| Неопределенный частный тип с тегами                          | Любой конкретный,       |
|                                                              | ограниченный тип с      |
| **Формат**: :ada:`type T (<>) is tagged private;`            | тегами, неопределенный  |
|                                                              | или определенный        |
+--------------------------------------------------------------+-------------------------+
| Неопределенный ограниченный частный тип с тегами             | Любой конкретный тип с  |
|                                                              | тегами, ограниченный или|
| **Формат**: :ada:`type T (<>) is tagged limited private;`    | нет, неопределенный или |
|                                                              | определенный            |
+--------------------------------------------------------------+-------------------------+
| Неопределенный абстрактный частный тип с тегами              | Любой неограниченный тип|
|                                                              | с тегами, неопределенный|
| **Формат**: :ada:`type T (<>) is abstract tagged private;`   | или определенный,       |
|                                                              | абстрактный или         |
|                                                              | конкретный              |
+--------------------------------------------------------------+-------------------------+
| Неопределенный абстрактный ограниченный частный тип с тегами | Любой тип с тегами,     |
|                                                              | ограниченный или нет,   |
| **Формат**:                                                  | неопределенный или      |
| :ada:`type T (<>) is abstract tagged limited private;`       | определенный,абстрактный|
|                                                              | или конкретный          |
+--------------------------------------------------------------+-------------------------+
| Неопределенный производный тип с тегами                      | Любой тип с тегами,     |
|                                                              | производный от базового |
| **Формат**: :ada:`type T (<>) is new B with private;`        | типа B, неопределенный  |
|                                                              | или определенный        |
+--------------------------------------------------------------+-------------------------+
| Неопределенный абстрактный производный тип с тегами          | Любой тип с тегами,     |
|                                                              | производный от базового |
| **Формат**:                                                  | типа B, неопределенный  |
| :ada:`type T (<>) is abstract new B with private;`           | или определенный        |
|                                                              | абстрактный или         |
|                                                              | конкретный              |
+--------------------------------------------------------------+-------------------------+

Те же примеры могут также содержать дискриминанты. В этом случае, :ada:`(<>)`
заменяется списком дискриминантов, например: :ada:`(D: DT)`.
